# 关卡副本
在大多数关卡型的手机游戏中，会有许多关卡（副本）可以战斗，而且是一关一关的来进行，另外还会依照难度分为一般副本跟菁英副本。在崑崙镜当中，则称为“普通关卡”与“困难关卡”，必须先通过普通关卡一次，才会出现敌人差不多、但难度提高的同样关卡，也就是困难关卡。在崑崙镜当中，普通关卡有九章、困难关卡则有八章。<br>
